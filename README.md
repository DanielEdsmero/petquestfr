# PetQuest (Next.js App Router)

Client-side gamified productivity app scaffolded with Next.js + TypeScript + Tailwind.

## Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- lucide-react
- framer-motion
- sonner

## Run locally

```bash
npm install
npm run dev
```

Then open http://localhost:3000.
